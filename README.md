## 🚀 Featured Commercial Project

### 📦 Warehouse Dispatch System

Designed and developed a commercial warehouse dispatch solution used internally within the company.

The system streamlines warehouse operations by allowing staff to create dispatch dockets, allocate stock, scan barcodes and print dispatch labels directly from Android devices.

#### Backend

- Java
- Spring Boot
- Maven
- REST API
- Microsoft SQL Server
- Spring JDBC / JdbcTemplate
- Spring Data JPA
- Native SQL Queries

The backend manages:

- Dispatch docket creation
- Scanner allocation workflows
- Barcode lookups
- Stock lookups
- Sales order lookups
- Shipping and consignment processing
- User management
- Authentication endpoints

#### Android Application

Built as a native Android application using modern Android development technologies.

Technologies:

- Kotlin
- Jetpack Compose
- Retrofit
- OkHttp
- Moshi
- Gson
- Kotlin Coroutines
- Android Navigation

Features:

- Barcode scanner integration
- Scanner broadcast support
- Stock allocation workflows
- Dispatch docket management
- Sales order processing
- Shipping and consignment management
- Zebra printer integration
- TCP ZPL label printing
